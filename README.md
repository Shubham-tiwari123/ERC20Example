# ERC20Token Example

This project is built using truffle pet-box.


## Pre-Requisite:
1. Truffle
2. ganache-cli
3. npm

## Quick Start

1. clone the repository.
2. cd ERC20Example
3. npm install
4. truffle develop
5. compile
6. migrate --reset
7. test
